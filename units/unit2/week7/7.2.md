# Control Systems: Components, Loops, and Applications

## Learning Objectives
- Define input, process, output, feedback
- Differentiate open‑loop vs closed‑loop
- Describe controllers, sensors, actuators, transducers
- Explain control algorithms in automation

---

## Fundamentals
Components: Input → Process (control logic) → Output, with Feedback comparing output to desired target (closed‑loop).

---

## Open‑Loop vs Closed‑Loop

| | Open‑Loop | Closed‑Loop |
|---|---|---|
| Feedback | None | Continuous feedback |
| Pros | Simple, low cost | Accurate, adaptive |
| Cons | Not adaptive | More complex |
| Examples | Timer microwave, fixed irrigation | Thermostat, autopilot |

---

## Key Components
- Controller: microcontrollers/PLCs/PID
- Sensors: temperature, motion, pressure, etc.
- Actuators: motors, hydraulics/pneumatics, solenoids
- Transducers: energy conversion (e.g., microphone, piezo)
- Control algorithms: PID, fuzzy logic, on‑off

## Key Points
- Control systems regulate devices using input/process/output/feedback
- Closed‑loop improves accuracy via continuous adjustment
- Controllers, sensors, actuators, transducers are core building blocks
- Algorithms determine system behavior
