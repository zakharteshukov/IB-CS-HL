Introduction to Compression 

Definition: Compression is the process of reducing the size of data files to save storage space and improve transmission speed. 

Importance of Compression: 

  • Reduces file storage requirements.
  • Speeds up data transmission over networks.
  • Improves efficiency in digital communications.
  • Reduces bandwidth consumption in streaming and online applications. 

Lesson

Types of Compression 

Data compression can be classified into two main types: lossless and lossy. 

Lossless Compression  Lossy Compression 

Definition

Lossless compression reduces file size without losing any information.

Lossy compression reduces file size by permanently removing some data, which may slightly degrade quality.

Mechanism

Original data can be reconstructed exactly as it was.

Eliminates redundant or less important information to achieve higher compression rates.

Examples

ZIP files, PNG images, FLAC audio.

PEG images, MP3 audio, MP4 video.

Use Cases

Used where data integrity is critical, such as text documents, software programs, and medical imaging.

Used where a small loss of quality is acceptable, such as multimedia applications and web images.

Compression Techniques 

Run-Length Encoding (RLE)   Transform Coding  

Concept

RLE is a simple lossless compression technique that encodes consecutive repeated values as a single value and a count.

Transform coding is a lossy compression method that converts data into a different mathematical representation and removes less important details.

Example

  • Original data: AAAAABBBBCCCCCCDD
  • RLE compressed: A5B4C6D2

JPEG image compression uses the Discrete Cosine Transform (DCT) to represent image data more efficiently, removing high-frequency components that the human eye is less sensitive to.

Applications

Used in simple image formats (e.g., BMP, TIFF) and fax transmissions. 

Used in JPEG image compression, MP3 audio compression, and MPEG video encoding. 

Summary

Key Points to remember

  • Compression reduces file size to save space and improve efficiency.
  • Lossless compression maintains full data integrity, while lossy compression sacrifices some quality for higher compression ratios.
  • Run-length encoding is a simple lossless technique, while transform coding is a widely used lossy method for multimedia data.
