# IB Computer Science HL - Course Materials

This repository contains all materials for the IB Computer Science Higher Level course (via Pamoja).

## üìÅ Folder Structure

### `units/`
Main course content organized by units and weeks.
- **unit2/**, **unit3/**, etc. - Each unit contains:
  - `unitN.md` - Unit overview
  - `weekN/` - Weekly lesson folders with:
    - `N.0.md` - Week overview
    - `N.1.md`, `N.2.md`, etc. - Individual lesson notes
  - `.html/` - Original HTML lesson files (backup)
  - `week10/` - Code examples and assignments

### `resources/`
Reference materials and official IB documents.
- **IBO/** - Official IB CS HL guides and subject briefs
- **.attachments/** - Images and media files
- `IBCSHLCoursebook.pdf` - Course textbook (if available)

### `topics/`
Topic-based notes organized by IB syllabus structure.
- **A/** - Topic A notes (e.g., A.1.1, A.1.2, etc.)
- Follows the IB CS HL syllabus organization

### `documents/`
Weekly PDF documents and handouts from Pamoja.
- `W2 - A1.1.1-A1.1.5.pdf` - Week 2 materials
- `W3 - A1.1.6 to A1.1.9.pdf` - Week 3 materials
- etc.

### `drawings/`
Excalidraw diagrams and visual notes.
- `.excalidraw.md` files - Interactive diagrams
- `.base` files - Drawing base files

### `archive/`
Old backups and archived materials (gitignored).

## üöÄ Quick Navigation

### Find Course Content
- **Current unit**: Check `units/unit2/` or `units/unit3/`
- **This week's lessons**: Navigate to `units/unitN/weekN/`
- **Week overview**: Read `N.0.md` in the week folder

### Find Resources
- **IB Guides**: `resources/IBO/`
- **Images**: `resources/.attachments/`
- **PDFs**: `documents/`

### Find Topic Notes
- **Topic A.1.1**: `topics/A/A.1/A.1.1/A.1.1.md`

## üìù File Naming Conventions

- **Units**: `unitN.md` (e.g., `unit2.md`)
- **Weeks**: `weekN/` folders
- **Lessons**: `N.0.md` (overview), `N.1.md`, `N.2.md`, etc.
- **Topics**: Follow IB syllabus (e.g., `A.1.1.md`)
